# Global-Sales-Analysis-project-2010-2016
This dashboard dives into global sales data from 2010 to 2016, highlighting:
•Total Orders
•Global Reach (No of Countries)
•Total Revenue
•Profit
•Units Sold

Key insights:
•Top-performing product categories
•Profit trends over the years
•Revenue by country (with map visuals )
•Online vs Offline order breakdown
•Filters for region, country, priority, and item type

Skills practiced
Data storytelling , Interactive visual design, KPI reporting ,Mapping , Drill down filters
